# GameVault

## Опис

**GameVault** — це десктопний застосунок для управління особистою бібліотекою відеоігор.

Програма дозволяє користувачу переглядати список своїх ігор, відстежувати прогрес досягнень та керувати інформацією про ігри.

Застосунок розробляється мовою **C++** з використанням фреймворку **Qt**.

---

## Основні можливості

- перегляд бібліотеки ігор
- відображення інформації про гру
- відстеження досягнень
- пошук ігор у бібліотеці
- додавання та видалення ігор
- імпорт даних із Steam API

## План розробки

Проєкт розробляється протягом одного місяця та поділений на декілька етапів:

- **1 тиждень** — планування та проектування архітектури програми
- **2 тиждень** — реалізація основної програмної логіки
- **3 тиждень** — розробка графічного інтерфейсу
- **4 тиждень** — інтеграція та тестування застосунку

## Команда

### Tech Lead / UI Developer

Відповідає за:

- проектування архітектури застосунку
- розробку графічного інтерфейсу у Qt
- реалізацію взаємодії користувача з програмою
- інтеграцію GUI з програмною логікою

### Backend Developer

Відповідає за:

- реалізацію програмної логіки
- створення моделей даних (`Game`, `Achievement`)

- інтеграцію з Steam API
